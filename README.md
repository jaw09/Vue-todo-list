建立時間： 6/11/18

## 網站實例

  [Vue Todo List](http://vue-todolit.s3-website-ap-northeast-1.amazonaws.com/)

## 實現工具

1. Vue
2. Vue-cli
3. Vuex
4. Axios
5. Firebase
6. Bootstrap4
7. Amazon S3

## Quickstart

``` bash
# 下載專案
git clone https://github.com/jaw09/Vue-todo-list.git

# 安裝套件
npm install

# 在本地段瀏覽專案
npm run dev

# 將專案打包輸出
npm run build
```
